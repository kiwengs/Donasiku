## Cara Menjalankan

1. Pindahkan folder project ke `C:\xampp\htdocs`.
2. Jalankan XAMPP.
3. Aktifkan Apache dan MySQL.
4. Import database dari `database/db_donasiku.sql`.
5. Buka browser ke `http://localhost/Donasiku-main`.

## Struktur Folder

- `api/` berisi endpoint backend.
- `component/` berisi header dan footer yang dipakai beberapa halaman.
- `database/` berisi file SQL database.
- `docs/` berisi dokumentasi project/API.
- File `.php` di root adalah halaman utama aplikasi.
